# Parabank Hackathon – Selenium Automation

## 📌 Overview
Automation framework created for the **Parabank online banking application** as part of a hackathon.  
Built with **Selenium WebDriver + TestNG + Cucumber (BDD)**.

## 🛠️ Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Cucumber (BDD)  
- Page Object Model (POM)  
- Utilities for data/config handling  

## 🚀 Features
- Hybrid test framework with both **TestNG** and **Cucumber**.  
- Implemented Page Object Model for maintainable, reusable code.  
- Feature files in Gherkin syntax.  
- Automated workflows: login, account summary, fund transfer, and error validations.  
- Screenshots captured on failures.  

## 📂 Structure
- `/features` → Cucumber feature files (Gherkin)  
- `/pages` → Page Object classes  
- `/steps` → Step definitions  
- `/tests` → TestNG tests  
- `/utils` → helper utilities (Excel, config, etc.)

## 🔗 How to Run
1. Clone repo.  
2. Import into Eclipse/IntelliJ.  
3. Run via TestNG XML or Cucumber runner.
